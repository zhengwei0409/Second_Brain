## Key Lessons / Takeaways

- AI makes writing code much faster
- But testing, code review, version control, etc. were built for the old, slower speed
- They don't speed up automatically just because code-writing did
- So new problems show up — often in places you didn't expect
- Don't just ask "how do I code faster with AI?"
- Ask: "if output suddenly went up 10x, what breaks first?"
