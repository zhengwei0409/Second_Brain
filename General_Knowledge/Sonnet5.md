# Sonnet 5 的 5 个重点总结：

1. 更强的 agentic 能力 — 能自主 make plans、使用 browser 和 terminal 等 tools，完成复杂多步骤任务，不再像以前那样中途卡住。
2. 性能接近 Opus 4.8，但价格更低 — 在很多 benchmark 上逼近旗舰模型 Opus 4.8 的水平，knowledge work 方面甚至略微超过它。
3. 价格便宜约 60% — Introductory pricing 是 $2/M input、$10/M output（到 2026 年 8 月 31 日），之后涨到 $3/$15，比 Opus 4.8 便宜不少。
4. 默认模型 — 现在是 Free 和 Pro 用户的 default model，同时也支持 Max、Team、Enterprise，以及 Claude Code 和 API（claude-sonnet-5）。
5. 更安全 — 相比 Sonnet 4.6，undesirable behaviors、hallucination、sycophancy 的比例都更低，但 cybersecurity 相关能力仍明显弱于 Opus 系列。
