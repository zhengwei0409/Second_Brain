# 2.3 — The Personal Computing Era (1980s)

## 1. High-Level Map（这个主题在大局中的位置）

上一节（[[2.2 — The Mainframe Era (1950s–1970s)]]）我们讲了一个很"憋屈"的世界：计算机是**机构专属**的巨兽，一般人连摸都摸不到，软件更是连"卖"都不能单独卖——它只是硬件的赠品。

那一节的结尾埋了个伏笔：**"软件产业"作为独立商业模式，要等到 PC 时代才真正诞生。**

这就是 2.3 要讲的故事。1980 年代发生了一件事，直接把"谁能拥有计算机"这个问题的答案，从"只有大机构"变成了"任何一个普通家庭、普通人"。这不只是硬件变小变便宜这么简单——它彻底改写了软件这门生意的玩法，也第一次创造出"软件公司"这种东西可以靠卖磁盘赚钱、上市、成为世界首富摇篮的可能性（没错，说的就是 Bill Gates）。

在整条历史时间线上，2.3 是**从"机构软件"到"消费软件"的转折点**，也是理解后面 2.4（互联网时代）、2.5（移动云时代）为什么会长成那样的地基。

---

## 2. Fundamentals First（从零开始，每个术语都讲透）

### 2.1 什么是 Personal Computer（个人电脑，PC）？

先从字面意思理解：**Personal**（个人的）+ **Computer**（计算机）。

这个词今天听起来平平无奇，但在 1970 年代末，这四个字是**革命性**的组合。因为在那之前，"计算机"和"个人"这两个词根本不会出现在同一个句子里——上一节讲过，计算机是几百万美元一台、占满一整个房间、只有银行政府才买得起的庞然大物。

那 1970 年代末到 1980 年代发生了什么，让"个人拥有一台计算机"变成可能？

关键是**微处理器（microprocessor）**技术的进步——简单说，就是把原本要塞满一整个房间的运算电路，压缩到一块小小的芯片上。这让计算机可以做到：
- 体积：从"一个房间"缩小到"一张桌子"能放下
- 价格：从"几百万美元"降到"几千美元"（一个中产家庭攒钱能买得起）
- 使用者：从"整个机构共用一台"变成"一个人、一张桌子、一台电脑"

代表性的产品在这个年代密集出现：Apple II（1977）、IBM PC（1981）、后来的 Macintosh（1984）。这些名字你可能都听过，但现在你知道了它们真正的意义——**它们是第一批把计算机从"机构的仆人"变成"个人的工具"的产品。**

**类比**：如果说 Mainframe 时代是"整个城市只有一辆巴士，所有人排队上车"，那 PC 时代就是**汽车开始量产、走进千家万户**——福特 Model T 那种"人人都能拥有一辆车"的历史时刻，只是这次的主角是计算机。

### 2.2 Shrink-Wrapped Software（收缩膜包装软件）——软件第一次变成"商品"

这是本节**最关键的概念**，因为它直接回答了上一节留下的伏笔："软件产业到底是怎么独立出来的？"

**Shrink-wrapped software** 字面意思是"用收缩膜包起来的软件"——想象你走进商店，货架上摆着一个个盒子，里面装着软盘（floppy disk）或后来的光盘，外面用塑料膜封装好，就跟你买一盒游戏卡带或一本书一样。

这在今天听起来毫无稀奇，但你要对比着上一节理解，才会明白它有多颠覆：

| | Mainframe 时代 | PC 时代 |
|---|---|---|
| 软件怎么来 | 跟着硬件"免费赠送" | **单独包装、单独定价、单独销售** |
| 谁写软件 | 机构内部的程序员，为自己单位定制 | 独立公司写好，卖给**任何**买了 PC 的人 |
| 卖多少份 | 通常只给自己机构用一次 | 同一份软件可以**复制无数次**，卖给成千上万人 |

这里有个经济学上的巨大转变：软件第一次变成了一种**可以大规模复制、边际成本极低**的商品。写一次 Word 处理软件，复制一万份磁盘去卖，成本几乎不会因为多卖一份而显著增加。这个特性，就是后来整个软件产业能够创造出巨额财富的根本原因。

### 2.3 Microsoft & Apple 的崛起——两种不同的打法

1980 年代最具代表性的两家公司，恰好代表了两种完全不同的商业策略：

**Microsoft**：选择只做**软件**，不做硬件。他们把 MS-DOS（后来是 Windows）操作系统卖给各种不同品牌的 PC 制造商（IBM、Compaq 等等）去装在自己的机器上。这个策略让 Microsoft 的软件几乎装在了当时市面上**所有**的 PC 里——你不用买 Microsoft 牌的电脑，也在用 Microsoft 的软件。

**Apple**：选择软硬件**一起做**。Apple 自己设计电脑（硬件），也自己写操作系统和软件，两者深度绑定、无法分家。这也是为什么 Apple 的产品体验通常更"一体化"，但价格也更贵、市场占有率长期不如 Microsoft 阵营广。

这两种打法，其实到今天都还在延续——你可以在后面章节看到，Google Android（像 Microsoft 的开放授权模式）vs Apple iOS（像 Apple 自己的封闭一体模式），本质上是同一场商业哲学之争的延续。

### 2.4 Consumer Software Market（消费软件市场）的诞生

因为 PC 走进了千家万户，"软件的客户"这个概念也彻底变了。

- **Mainframe 时代**：软件的客户是"某个机构的 IT 部门"——一次性谈判、定制化开发。
- **PC 时代**：软件的客户是"数以百万计、素未谋面的普通消费者"——你不知道买你软件的人是谁，你只是把盒子摆上货架，等人来买。

这催生了一整套全新的商业模式：文字处理软件（word processor）、电子表格（spreadsheet，比如 Lotus 1-2-3）、游戏，这些都是冲着"普通人会不会想买"去设计的，而不是"某个银行需不需要"。

这也是"消费者产品思维"第一次进入软件世界——你现在熟悉的"产品经理要研究用户需求"这种思维方式，根子就在这里。

### 2.5 Independent Software Vendor（ISV，独立软件供应商）

最后一个概念，帮你把整节课串起来：**ISV** 指的是那些**不生产硬件、专门开发和销售软件产品**的公司。

在 Mainframe 时代，这种公司几乎不存在——因为软件是硬件的附属品，没有独立存在的意义。而 PC 时代因为"软件可以单独打包销售"（2.2 讲的 shrink-wrapped software），才第一次让"一家公司只靠卖软件就能活下去、甚至发大财"变成现实。Microsoft 本身，就是史上最成功的 ISV 之一。

这个词今天可能你不常听到了（现代说法更常是"软件公司"或"SaaS 公司"），但它的历史意义很重要：**它标志着"软件"正式从"某个产品的附属服务"，升级成了"一门独立的生意"。**

## 3. Progressive Explanation（把这些概念串起来）

现在把这几块拼图拼起来，讲一个完整的故事：

> 1970 年代末，**微处理器**技术让计算机第一次能被压缩成一台桌上能放、普通家庭买得起的机器——**Personal Computer（PC）** 就此诞生。Apple II、IBM PC 这些产品，把计算机的使用者从"机构"变成了"个人"。
>
> 因为 PC 走进了千家万户，软件的"客户"也从"某个机构的 IT 部门"变成了成千上万个素未谋面的普通消费者——这就是 **Consumer Software Market** 的诞生。
>
> 而要服务这么多分散的消费者，软件不可能再像 Mainframe 时代那样"跟着硬件免费赠送、一次性定制"了。软件开始被打包成一个个 **Shrink-Wrapped Software** 产品，摆上商店货架，像卖书、卖唱片一样卖给任何走进店里的人。
>
> 这种"软件可以独立打包销售"的模式，第一次让专门只做软件、不做硬件的公司能够生存壮大——这就是 **Independent Software Vendor（ISV）**。**Microsoft** 和 **Apple** 正是这个时代崛起的代表：一个专攻软件、授权给所有硬件厂商用；一个软硬件一体，打造封闭但一体化的体验。两种打法都证明了一件事——**软件，终于成为了一门可以独立存在、独立赚钱的生意。**

## 4. 5 个你必须完全掌握的核心概念

### ① Personal Computer（个人电脑，PC）
- **定义**：一台体积小、价格亲民、供单一个人使用（而非整个机构共享）的计算机。
- **为什么重要**：它是整个 2.3 时代的物理基础——没有 PC 变得便宜、变得小，后面"软件卖给消费者"这整套商业模式根本无从谈起。
- **自测题**：PC 和 Mainframe 在"使用者"这一点上最核心的区别是什么？
  **答案**：Mainframe 是由整个机构共享使用的昂贵设备；PC 是价格亲民到可以由一个人独立拥有和使用的设备。

### ② Shrink-Wrapped Software（收缩膜包装软件）
- **定义**：把软件做成实体商品（磁盘/光盘装在盒子里，用塑料膜封装），像普通商品一样在货架上单独销售的模式。
- **为什么重要**：这是软件第一次从"硬件的附属赠品"变成"可以独立标价、独立销售、大规模复制"的商品，是整个软件产业能独立赚钱的起点。
- **自测题**：为什么 Shrink-Wrapped Software 的出现，是软件产业史上的一个转折点？
  **答案**：因为在此之前软件是随硬件免费赠送的服务；而 shrink-wrapped software 让同一份软件可以低成本复制成千上万份、单独打包卖给不同的消费者，这让软件第一次可以作为独立商品创造利润。

### ③ Microsoft & Apple 的崛起
- **定义**：1980 年代两家最具代表性的 PC 时代公司，分别代表"只做软件、授权给所有硬件厂商"（Microsoft）和"软硬件一体、封闭生态"（Apple）两种商业策略。
- **为什么重要**：这两种商业哲学的对立，一直延续到今天（例如 Android vs iOS），理解这个源头能帮你看懂现代科技公司竞争格局的历史根源。
- **自测题**：Microsoft 和 Apple 在 1980 年代的核心商业策略有什么不同？
  **答案**：Microsoft 只做软件（操作系统），把它授权给各种不同品牌的硬件厂商使用；Apple 则软硬件都自己做，两者深度绑定，不对外授权。

### ④ Consumer Software Market（消费软件市场）
- **定义**：以数以百万计的普通个人消费者（而非机构客户）为目标的软件销售市场。
- **为什么重要**：它标志着软件开发的思维方式从"满足某个机构的定制需求"转变为"猜测和满足普通大众的需求"，是今天"产品思维"和"用户体验"这些概念的历史起点。
- **自测题**：消费软件市场和 Mainframe 时代的软件"客户"有什么本质区别？
  **答案**：Mainframe 时代软件的客户是某个特定机构的 IT 部门，通常是一次性定制;消费软件市场的客户是成千上万个互不相识的普通个人消费者，软件要能大规模、标准化地卖给他们。

### ⑤ Independent Software Vendor（ISV，独立软件供应商）
- **定义**：不生产硬件、专门开发和销售软件产品来维持生计的公司。
- **为什么重要**：ISV 的出现证明了"软件"可以脱离硬件独立存在，成为一门自给自足的生意——这是"软件公司"这个商业物种诞生的标志，Microsoft 就是最典型的例子。
- **自测题**：为什么 ISV 这种公司在 Mainframe 时代几乎不可能存在，但在 PC 时代却能蓬勃发展？
  **答案**：因为 Mainframe 时代软件是硬件的免费附属品，没有独立的商业价值；而 PC 时代有了 shrink-wrapped software 这种可以独立打包销售的模式，软件本身就能创造收入，让专门只做软件的公司也能生存壮大。
